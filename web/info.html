Hoe werkt het?

Structuur van reeksen, wedstrijden en klassen

Aan de slag als beheerder

1

Maak een reeks

Een reeks groepeert wedstrijden, bijv. een hele zomercompetitie. Geef het een naam, seizoen en omschrijving.

2

Voeg klassen toe aan de reeks

Reeksklassen gelden voor álle wedstrijden in de reeks. Elke klasse krijgt automatisch een unieke 6-letter deelnamecode.

3

Voeg wedstrijden toe

Plan wedstrijden binnen de reeks met naam en datum. Of maak losse eenmalige wedstrijden.

4

Deel de codes

Geef de deelnamecode aan deelnemers. Zij voeren die in de app in, waarna hun GPX-track automatisch aan de juiste klasse wordt gekoppeld.

Wat is wat?

Reeks — de paraplu. Een reeks bundelt meerdere wedstrijddagen, bijvoorbeeld een woensdagavondcompetitie over een heel seizoen.

Reeksklassen zijn het handigst voor competities met vaste klassen (A, B, C). De code blijft het hele seizoen geldig — deelnemers hoeven maar één keer hun code in te voeren. De server koppelt elke track automatisch aan de dichtstbijzijnde wedstrijddatum.

Wedstrijd — één racedag. Kan bij een reeks horen, of op zichzelf staan (losse wedstrijd).

Wedstrijdklassen — klassen die alleen voor díe specifieke wedstrijd gelden. Handig voor eenmalige evenementen met afwijkende indeling.

Deelnamecodes zijn 6-letter codes zonder verwarrende tekens (geen O/0, I/L). Elke klasse — reeks of wedstrijd — krijgt er automatisch één. Een wedstrijd kan zowel reeksklassen als wedstrijdklassen hebben; de code bepaalt in welke klasse een track valt.

Voorbeeld

Stel, je organiseert de Woensdagavondcompetitie 2026:

  • Je maakt de reeks "Woensdagavond competitie" met seizoen "2026"
  • Je voegt reeksklassen toe: Klasse A (code K4X9BT) en Klasse B (code P7M2ND)
  • Je voegt 8 wedstrijden toe, elke woensdag van mei t/m september
  • Deelnemers in klasse A krijgen code K4X9BT, voeren die één keer in
  • Elke woensdag uploadt de app hun GPX-track — de server koppelt hem automatisch aan de juiste wedstrijd en klasse

Op het dashboard zien deelnemers hun tracks, en kunnen ze boten vergelijken op de kaart.
